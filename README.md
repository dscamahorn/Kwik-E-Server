# Kwik-E-Server
Spin up a live-reload localhost...real kwik!

**Install**
```
npm install
```

**Configure**
```js
var projectPath = ['./www/*.+(js|html|css)', './www/**/*.+(js|html|css)', './www/**/**/*.+(js|html|css)'];
```

**Run**
```
gulp
```

## Dependencies
* Node.js - https://nodejs.org
* npm - https://www.npmjs.com/
* Gulp - http://gulpjs.com/
